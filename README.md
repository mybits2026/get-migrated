# Get Migrated - PR Readiness Concierge

This project is a lean, V1 MVP for a PR Readiness Concierge service. It unbundles the migration agent process by offering affordable, reliable preparation and planning for Nepali students/graduates in Australia before they engage a licensed migration expert.

## Tech Stack
- **Framework:** Next.js (App Router)
- **UI/Components:** React 19, `@base-ui/react`, Radix/Shadcn primitives
- **Styling:** Tailwind CSS v4
- **Language:** TypeScript

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open http://localhost:3000 in your browser.

## Project Structure

- `src/app/`: Routes and pages (currently a single page MVP).
- `src/components/ui/`: Reusable UI components.
- `src/config/`: Site configuration and global metadata.
- `src/lib/`: Shared utilities.
- `DESIGN.md`: Design system rules and color definitions.
